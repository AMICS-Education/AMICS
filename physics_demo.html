Mechanics

Motion in One Dimension

The study of motion without external agents is called kinematics. There are three main types of motion: translational, rotational, and vibrational. For simplicity’s sake, we are going to treat objects as particles: they have mass, but are infinitesimally small.

Position, Velocity, and Speed

A particle’s position is its location relative to a reference point. Its displacement is defined as the change in position in some time interval. As the particle moves from \(x_1\) to \(x_2\), its displacement is: \[\Delta x = x_2 - x_1\]

The Greek character \(\Delta\) (Delta) generally means difference. We also have the concept of velocity, which is the change in position with respect to time. Mathematically, that is: \[v_{x,\text{avg}} = \dfrac{\Delta x}{\Delta t}\]

This is the average velocity over a period of time. We can see that its sign depends on the displacement, because \(t > 0\). But there’s also speed, which depends on distance, so it’s always positive: \[v_{\text{avg}} = \dfrac{d}{\Delta t}\]

  • Displacement is a vector quantity, but distance is a scalar quantity.

  • Velocity is a vector quantity, but speed is a scalar quantity.

Instantaneous Velocity and Speed

This is where calculus jumps in. We take the limit as the intervals of time go to zero, and we acquire: \[v_x = \lim_{\Delta t \rightarrow 0} \dfrac{\Delta x}{\Delta t} = \dfrac{\dd x}{\dd t}\]

The derivative of displacement with respect to time is velocity.

Analysis Models: The Particle Under Constant Velocity

We’ll try to describe the particle. We know that the velocity is constant, so the only thing to find out here is its displacement. Luckily, this is simple. Rearranging (2) gives: \[x_2 = x_1 + v_x t\] These analysis models get tougher over time.

Acceleration

Analogously, we define acceleration as the change in velocity over time. Then we have the equation of average acceleration: \[a_{x, \text{avg}} = \dfrac{\Delta v_x}{\Delta t}\] and similarly, the instantaneous acceleration: \[a_x = \lim_{\Delta t\rightarrow 0} \dfrac{\Delta v_x}{\Delta t} = \dfrac{\dd v}{\dd t}\]

The derivative of velocity with respect to time is acceleration.

In 1D kinematics, if the velocity and acceleration are pointing in the same direction, then the object is speeding up and vice versa.

The Particle Under Constant Acceleration

Let’s tackle the most simple case. Because acceleration is the derivative of velocity, so if it’s a constant, then the velocity changes linearly. We can write: \[v_{x_2} = v_{x_1} + a_x t\] And if velocity is a linear function, then that means displacement will be a quadratic. Integrating both sides give: \[x_2 = x_1 + v_{x_1} t + \dfrac{1}{2}a_x t^2\] We can even extract velocity as a function of position. We know: \[v_{x_2} = v_{x_1} + a_x t\] \[x_2 = x_1 + \left( \dfrac{v_{x_1} + v_{x_2}}{2}\right) t\] Substitute \(t\) in gives: \[x_2 - x_1 = \dfrac{(v_{x_1} + v_{x_2})(v_{x_2} - v_{x_1})}{2a_x}\] And we have the final form: \[v_{x_2}^2 - v_{x_1}^2 = 2a_x d\]

We have 4 important equations:

  1. \(v_{x_2} = v_{x_1} + a_x t\)

  2. \(x_2 = x_1 + \left( \dfrac{v_{x_1} + v_{x_2}}{2}\right) t\)

  3. \(x_2 = x_1 + v_{x_1} t + \dfrac{1}{2}a_x t^2\)

  4. \(v_{x_2}^2 - v_{x_1}^2 = 2a_x d\)

Free Fall

In free fall, objects undergo constant acceleration \(g \approx 9.8 \mathrm{m/s^2}\), and starting velocity \(v_{x_1} = 0\). Then we have a series of equations:

  • \(v = gt\)

  • \(t = \sqrt{\dfrac{2h}{g}}\)

  • \(v_{\text{final}} = \sqrt{2gh}\)

Vectors

Vectors are born because we need to study quantities that have both magnitude and direction. To describe them mathematically, we employ the use of coordinate systems.

Coordinate Systems

In 2D, there are 2 main types of coordinate systems: The Cartesian system \((x,y)\) and the polar system \((r,\theta)\). It’s worth knowing that \(r = \sqrt{x^2 + y^2}\) and \(\theta = \arctan{\dfrac{y}{x}}\).

Vector and Scalar Quantities

It’s all about if the quantities are directional or not.

  • A scalar quantity is completely specified by a single value with an appropriate unit and has no direction.

  • A vector quantity is completely specified by a number and appropriate units plus a direction.

We denote a vector as boldface characters, like \(\mathbf{A}\), or \(\vec{a}\). Their magnitude is given by \(|\mathbf{A}|\) or \(|\vec{a}|\)

Some Properties of Vectors

They obey commutativity and associativity under addition. The negative of a vector \(\mathbf{A}\) is \(\mathbf{-A}\), and their sum is 0. The negative vector has the same magnitude, but it points in the opposite direction.

Subtracting \(\mathbf{A - B}\) is the same as \(\mathbf{A + (-B)}\). When multiplying a vector by a scalar, only the magnitude changes. The direction stays the same.

Components of a Vector and Unit Vectors

We can split a vector along the chosen axes. Writing \(\mathbf{A} = \mathbf{A}_x + \mathbf{A}_y\) and \(\mathbf{B} = \mathbf{B}_x + \mathbf{B}_y\), we can quickly calculate the resultant vector: \(\mathbf{R} = (\mathbf{A+B})_x + (\mathbf{A+B})_y\). The resultant magnitude and direction are obtained similarly to how we convert from Cartesian to polar.

More commonly, we see people write unit vectors: \[\mathbf{A} = x\mathbf{\vec{i}} + y\mathbf{\vec{j}} + z\mathbf{\vec{k}}\]

They’re essentially splitting the vectors into its components and making sure that the magnitude is 1.

Motion in Two Dimensions

Now that we have the tools for 2D kinematics, we can study motion in two dimensions by splitting the vectors of motion into their components along the \(x\) and \(y\) axes, therefore reducing the complexity back to 1D (in some sense).

The Position, Velocity and Acceleration Vectors

Let’s describe the position of a particle as a vector \(\mathbf{\vec{r}}\), drawn from the origin to the particles position in the coordinate plane. Then we define the velocity vector \(\mathbf{\vec{v}}\) and as per the formulae established earlier, we have: \[\mathbf{\vec{v}} = \dfrac{\Delta\mathbf{\vec{r}}}{\Delta t}\] Because time is a scalar quantity, we deduce that velocity is a vector quantity directed along \(\Delta \mathbf{\vec{r}}\).

Remember, the velocity is independent of the path taken, as it only takes into account the displacement between the final and initial position. The speed, however, is calculated by the total length of the path divided over time.

Instantaneous velocity and acceleration is defined analogously, only this time we are dealing with vector-valued functions.
