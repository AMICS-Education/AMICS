Integral Calculus in One Variable

Indefinite Integrals

Integral of a function

The integral is the inverse of the derivative operator. Given a function \(f(x)\), we can find a function \(F(x)\) such that \(F'(x) = f(x)\). The integral of a function is not unique, as for each \(f(x)\), \(F(x) + C\) is also an integral of that function.

Both the derivative and the integral are linear operators, so we have: \[\int [\alpha f(x) + \beta g(x) ]\dd x = \alpha \int f(x)\dd x + \beta \int g(x) \dd x\]

Ways to calculate indefinite integrals

  1. Expansion

    We use the linearity rule to turn a complicated integral into the sum of many simpler ones, then calculate one by one.

  2. Changing the differential expression (\(u\)-substitution)

    If \(\displaystyle{\int f(x) \dd x = F(x) + C}\) then \(\displaystyle{\int f(u) \dd x = F(u) + C}\), where \(u = u(x)\) is a continuously differentiable function. Then, we can change the integrand \(g(x)\dd x\) into: \[g(x) = f(u(x))u'(x) \dd x\] Then the integral turns into: \[\int g(x) \dd x = \int f(u(x))u'(x) \dd x = \int f(u(x)) \dd u = F(u) + C\]

    In the simple case \(u = ax+b\), we have \(\dd u = a \dd x\). Then if \(\displaystyle{\int f(x) \dd x = F(x) + C}\) then: \[\int f(ax+b) \dd x = \dfrac{1}{a} F(ax+b) +C\]

  3. Change of variables

    Consider the integral \(\displaystyle{I = \int f(x) \dd x}\), where \(f(x)\) is a continuous function. We can change \(f(x)\) such that we work with functions with known or easier antiderivatives:

    1. Change of variables type 1:

      Let \(x = \varphi(t)\), where \(\varphi(t)\) is a monotonic and continuously differentiable function. Then: \[I = \int f(x) \dd x = \int f[\varphi(t)]\varphi'(t) \dd t\] Denote the antiderivative of \(g(t) = f[\varphi(t)]\varphi'(t)\) as \(G(t)\) and \(h(x)\) as the inverse of \(x = \varphi (t)\), we then have: \[\int g(t) \dd t = G(t) + C \Rightarrow I = G[h(x)] + C\]

    2. Change of variables type 2:

      Let \(t = \psi(x)\), where \(\psi(x)\) is a continuously differentiable function and we can write \(f(x) = g[\psi(x)]\psi ' (x)\). Then: \[I = \int f(x) \dd x = \int g[\psi(x)]\psi'(x) \dd x\]

      Denote the antiderivative of \(g(t)\) as \(G(t)\), then: \[I = G[\psi(x)] + C\]

      Remember to change back to the original variable!

  4. Integration by parts

    Let \(u = u(x)\) and \(v = v(x)\) be continuously differentiable functions. We know: \[\dd (uv) = u \dd v + v \dd u \Rightarrow \int \dd (uv) = \int u \dd v + v \dd u\] Then we have the following formula: \[\int u \dd v = uv - \int v \dd u\] Consider the integral \(I = \displaystyle{\int f(x) \dd x}\) We need to express: \[f(x) \dd x = [g(x)h(x)]\dd x = g(x) [h(x) \dd x] = u \dd v\] then apply the integration by parts formula to \(u = g(x)\), \(v = \displaystyle{h(x) \dd x}\)

Integral of rational functions

A rational function is one with the form \(f(x) = \dfrac{P(x)}{Q(x)}\), where \(P(x)\) and \(Q(x)\) are polynomials in \(x\). If \(\deg P(x) < \deg Q(x)\) then it’s called a true rational function.

Using polynomial division, we can rewrite any rational function: \[f(x) = H(x) + \dfrac{r(x)}{Q(x)}\]

The integral of \(H(x)\) can easily be computed. As for the true rational function \(\dfrac{r(x)}{Q(x)}\), we will use partial fractions to decompose it into four simpler types of functions. First, using the method of Undetermined Coefficients, we can write \(Q(x)\) as the product of linear polynomials and quadratic polynomials with no real roots: \[Q(x) = \prod_{k = 1}^{n}(x-a_k)^{\alpha_k} \prod_{k=1}^{n}(x^2+p_kx + q_k)^{\beta_k}\] Note that this can always be achieved as per the Fundamental Theorem of Algebra. Then, we just need to compute the following four types of integrals:

  1. \(\displaystyle{\int \dfrac{A}{x-a} \dd x}\)

  2. \(\displaystyle{\int \dfrac{A}{(x-a)^k} \dd x} \quad (k \geq 2)\)

  3. \(\displaystyle{\int \dfrac{Mx+N}{x^2 + px + q}\dd x}\)

  4. \(\displaystyle{\int \dfrac{Mx+N}{(x^2 + px + q)^m}\dd x \quad (m \geq 2)}\)

These integrals cover all cases because any partial fraction decomposition over \(\mathbb{R}\) involves repeated linear factors and irreducible quadratic factors, possibly raised to powers. The first two ones are simple:

  1. \(\displaystyle{\int \dfrac{A}{x-a}\dd x = A\ln|x-a| + C}\)

  2. \(\displaystyle{\int \dfrac{A}{(x-a)^k}\dd x = A\int (x-a)^{-k} = \dfrac{-A}{(k-1)(x-a)^{k-1}} \quad (k \geq 2)}\)

The third one is a bit more complicated: we need to use a clever substitution, inspired by completing the square, to bring it back to familiar forms:

Let \(a = \sqrt{q-\dfrac{p^2}{4}}\) and \(t = x + \dfrac{p}{2}\), the integral becomes: \[\begin{aligned} \int \dfrac{Mx+N}{x^2 + px + q}\dd x & = \int \dfrac{Mt + (N - \frac{Mp}{2})}{t^2 + a^2} \dd t \\ & = M \int \dfrac{t}{t^2 + a^2} \dd t + \left(N-\frac{Mp}{2} \right) \int \dfrac{1}{t^2 + a^2} \dd t \end{aligned}\] For the first integral, substitute in \(u = t^2\) and we have a logarithm. For the second one, divide both the numerator and the denominator by \(a^2\) then substitute \(u = \dfrac{t}{a}\), we yield the inverse tangent. Then the integral is simply: \[\int \dfrac{Mx+N}{x^2 + px + q}\dd x = \dfrac{M}{2}\ln(x^2 + px+ q) + \dfrac{2N - Mp}{\sqrt{4q-p^2}}\arctan \dfrac{2x+p}{\sqrt{4q -p^2}} + C\]

Before moving on to solve the last integral, let’s review a bit of complex numbers:

Some important (and beautiful) formulas:: \[\begin{aligned} \textbf{Euler's Formula:}\quad & e^{ix} = \cos x + i \sin x \\ \textbf{De Moivre's Formula:} \quad & (\cos x + i\sin x)^n = \cos nx + i \sin nx \end{aligned}\]

We will prove it using ODEs. Let \(f(x) = e^x\) and \(g(x) = \cos x + i \sin x\). We know: \[\begin{aligned} f'(x) & = ie^{ix} = if(x) \\ g'(x) & = -\sin x + i\cos x = ig(x) \end{aligned}\]

And also the initial values: \[f(0) = g(0) = 1\] By uniqueness of solutions to first-order linear ODEs, Euler’s formula is proven. By substituting \(f(nx) = g(nx)\), De Moivre’s formula is proven. Then we have:

  • \(\cos x = \Re(e^{ix}) = \dfrac{e^{ix} + e^{-ix}}{2}\)

  • \(\sin x = \Im(e^{ix}) = \dfrac{e^{ix} + e^{-ix}}{2i}\)

From this, we can derive a general formula for powers of the sine and cosine: \[\cos ^n x = \left(\dfrac{e^{ix}+e^{-ix}}{2}\right) ^n = \dfrac{1}{2^n} \sum_{k=0}^{n}\binom{n}{k}e^{i(n-2k)x}\] Now if we group \(k\) and \(n-k\) in pairs to get \(\cos(n-2k)\), we get: \[\cos ^n x = \dfrac{1}{2^n} \sum_{k=0}^{n} \binom{n}{k} \cos ((n-2k)x)\] Analogously, we also have: \[\sin ^n x = \dfrac{1}{(2i)^n} \sum_{k=0}^{n} \binom{n}{k} (-1)^k \cos (\dfrac{\pi n}{2}-(n-2k)x)\]

Now, back to the fourth integral, we first do the same thing: change variables \(a = \sqrt{q-\dfrac{p^2}{4}}\) and \(t = x + \dfrac{p}{2}\). This yields: \[\begin{aligned} \int \dfrac{Mx+N}{(x^2 + px + q)^m} \dd t= & \int \dfrac{Mt + (N - \frac{Mp}{2})}{(t^2 + a^2)^m} \dd t \\ = & M\int \dfrac{t}{(t^2 + a^2)^m} \dd t + \left(N-\dfrac{Mp}{2}\right) \int \dfrac{1}{(t^2 + a^2)^m} \dd t \end{aligned}\]

Again, to compute the first integral, we use the substitution \(u = t^2\). Then it is: \[M\int \dfrac{t}{(t^2 + a^2)^m} \dd t = \dfrac{-M}{2(m-1)(t^2 + a^2)^{m-1}} +C\]

For the second integral however, this time we need a different substitution. Let \(t = a\tan z\). Then: \(t^2 +a ^2 = a^2 \sec ^2 z\), and \(\dd t = a\sec ^2 \dd z\). Then the integral turns into: \[\int \dfrac{1}{(t^2 + a^2)^m} \dd t = a^{2m-1} \int \cos ^{2m-2} z \dd z\] Let \(I(z) = \displaystyle{\int \cos ^{2m-2} z \dd z}\)and now, we can take advantage of the linearity rule and the formula for powers of cosine we just derived. The integral turns into: \[\begin{aligned} I(z) = & \int \dfrac{1}{2^{2m-2}} \sum_{k=0}^{2m-2} \binom{2m-2}{k}\cos(2(m-k-1)z) \dd z \\ = & \dfrac{1}{2^{2m-2}} \sum_{k=0}^{2m-2} \binom{2m-2}{k} \dfrac{1}{2m-2-2k} \sin ( 2(m-k-1)z) +C \\ = & \dfrac{1}{2^{2m-2}} \left[ \binom{2m-2}{m-1}z + \sum_{k=0, k\neq m-1}^{2m-2} \binom{2m-2}{k} \dfrac{1}{2m-2-2k} \sin ( 2(m-k-1)z)\right] +C \end{aligned}\] Then the integral, as a whole, evaluates to:

\[\int \frac{Mx + N}{(x^2 + px + q)^m} \dd x = \frac{-M}{2(m - 1)(x^2 + px + q)^{m - 1}} + \left(N - \frac{Mp}{2} \right) \left( \sqrt{q - \dfrac{p^2}{4}}\right)^{2m-1}\cdot I(z) + C\]

Integral of trigonometric functions

  1. The general method

    Consider the integral \(\displaystyle{\int f(\sin x, \cos x)\dd x}\), where the integrand is a rational function in terms of \(\sin x\) and \(\cos x\). We can use the "universal trigonometric substitution" \(x = \tan \dfrac{t}{2}\). Then: \[\sin x = \dfrac{2t}{1+t^2}; \quad \cos x = \dfrac{1-t^2}{1+t^2}; \quad \tan x = \dfrac{2t}{1-t^2}; \dd t = \dfrac{2 \dd t}{1 + t^2}\] The integrand turns into a rational function in terms of \(t\).

  2. Integrals of the form \(\displaystyle{\int \sin ^m x \cos ^n x \dd x}\), where \(m,n\) are positive integers

    • If \(m\) is odd, we let \(t = \cos x\)

    • If \(n\) is odd, we let \(t = \sin x\)

    • If both \(m\) and \(n\) are even, we use power-reduction formulae: \[\sin ^2 x = \dfrac{1-\cos 2x}{2}; \quad \quad \cos ^2 x = \dfrac{1+\cos 2x}{2}\]

      Then we’ll have a similar integral with the form \(\displaystyle{\int \sin ^k 2x \cos ^l 2x \dd x}\)

  3. Special forms of \(\displaystyle{\int f(\sin x, \cos x) \dd x}\)

    • Let \(t = \cos x\) if \(f(-\sin x, \cos x) = - f(\sin x, \cos x)\)

    • Let \(t = \sin x\) if \(f(\sin x, -\cos x) = - f(\sin x, \cos x)\)

    • Let \(t = \tan x\) if \(f(-\sin x, -\cos x) = f(\sin x, \cos x)\)

Integral of irrational expressions

There are two main ways to solve these integrals: using trigonometric substitution, and using the Euler substitution. The trig-sub is very intuitive, but the Euler substitution is also very nice:

Let \(t = x + \sqrt{x^2 + a}\) for the integral \(\displaystyle{\int f(x, \sqrt{x^2 +a}) \dd x}\). Then: \[\dd t = 1 + \dfrac{x} {\sqrt{x^2 + a}} \dd x = \dfrac{x + \sqrt{x^2 +a }} {\sqrt{x^2 +a}} \dd x \Rightarrow \dfrac{\dd t}{t} = \dfrac{\dd x}{\sqrt{x^2 + a}}\]

Definite Integrals

Definition

Say \(f(x)\) is defined and bounded on \([a,b]\). Partition \([a,b]\) into \(n\) subintervals \([x_i, x_{i+1}]\) where \(a = x_0 < x_1 < \ldots < x_n = b\). In each interval \([x_i,x_{i+1}]\), we choose a point \(\xi \in [x_i, x_{i+1}]\) and form the expression \[S_n = \sum_{i=0}^{n-1}f(\xi_i) \Delta x_i\] where \(\Delta x_i = x_{i+1} - x_i\). Here, \(S_n\) is the Riemann sum. Denote \(\lambda = \displaystyle{\max_{1 \leq i \leq n}} \Delta x_i\). If there exists the limit \(I = \displaystyle{\lim_{\lambda \rightarrow 0} S_n}\) that doesn’t depend on how we partition \([a,b]\) and how we choose \(\xi_i\) then \(I\) is called the definite integral of the function \(f(x)\) on \([a,b]\), denoted \(\displaystyle{\int_a ^bf(x) \dd x}\). Then we say \(f(x)\) is integrable on \([a,b]\).

We then have defined the definite integral for all \(a<b\). We can then define, if \(b<a\), \(\displaystyle{\int_a ^bf(x) \dd x = - \int_b ^af(x) \dd x}\) and when \(a = b\), \(\displaystyle{\int_a ^bf(x) \dd x = 0}\).
